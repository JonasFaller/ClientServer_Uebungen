Die Übung 3 war schon etwas anspruchsvoller. Man musste vom Client einen Link schickn und der Server downloaded die Seite. 
Bei mir sendet der Client die Url als String zum Server. Der liest den Inhalt über die Url aus und sendet sie zurück zum Client.
Der Client speichert den Inhalt in einer Text Datei ab. 

Die Aufgabe funktioniert ohne Probleme.
